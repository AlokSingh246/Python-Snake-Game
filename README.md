# 🐍 Snake Game

A simple Snake Game built using **Python** and the **Turtle** graphics library. Control the snake, eat food to increase your score, and avoid hitting the walls or your own body.

## Features

- Snake movement using keyboard controls
- Random food generation
- Score and High Score tracking
- Snake grows after eating food
- Game resets on collision

## Technologies Used

- Python
- Turtle Graphics
- Random
- Time

## Controls

| Key | Action |
|-----|--------|
| W | Move Up |
| A | Move Left |
| S | Move Down |
| D | Move Right |

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/yourusername/Snake-Game.git
```

2. Open the project folder.

3. Run the game.

```bash
python Snake.py
```
